## Quick Start

### 1. Add Your Images

Place these images in `public/assets/`:

**Icons folder** (`public/assets/icons/`):
- `avatar-world-icon.png` (512x512px)

**Covers folder** (`public/assets/covers/`):
- `avatar-world-cover.png` (1200x630px)

**Root public folder** (`public/`):
- `favicon.ico` (32x32)
- `favicon-16x16.png`
- `favicon-32x32.png`
- `favicon-48x48.png`
- `apple-touch-icon.png` (180x180)
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`

### 2. Install Dependencies

```bash
npm install
```

### 3. Run Development Server

```bash
npm run dev
```

Open http://localhost:4321

### 4. Build for Production

```bash
npm run build
```

### 5. Deploy

Upload the `dist/` folder to:
- **Netlify** (Recommended)
- **Vercel**
- **Cloudflare Pages**
